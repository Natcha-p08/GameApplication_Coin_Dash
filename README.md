# GameApplication_Coin_Dash
Android Coin-Collector Racing Game > A dynamic 2D mobile racing game developed using Android Studio and Java. The project features real-time motion control/touch input, score tracking systems, and procedural coin spawning logic. 

Key Features:

- Smooth Gameplay Mechanics: Intuitive controls for vehicle maneuvering.

- Score System: Real-time coin collection and high-score tracking using Android SharedPreferences.

- Responsive UI: Optimized for various Android screen sizes using XML layouts.

- Asset Management: Integration of 2D sprites
